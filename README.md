# heroes-api
Gerenciador de super heróis da Marvel e da DC em uma API reativa com Spring Boot.

Esta é a solução do Projeto 2 do bootcamp TQI Java Developer na Digital Innovation One:
https://digitalinnovation.one/.

## Recursos utilizados:
- Java 8
- Spring Webflux
- Spring Data
- DynamoDB
- Junit
- slf4j
- Reactor
